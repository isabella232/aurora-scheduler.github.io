Design Documents

Since its inception as an Apache project, larger feature additions to the Aurora code base are discussed in form of design documents. Design documents are living documents until a consensus has been reached to implement a feature in the proposed form.

Current and past documents:

  • Command Hooks for the Aurora Client
  • Health Checks for Updates
  • JobUpdateDiff thrift API
  • REST API RFC
  • Revocable Mesos offers in Aurora
  • Ubiquitous Jobs

Design documents can be found in the Aurora issue tracker via the query project = AURORA AND text ~ "docs.google.com" ORDER BY created.
